jQuery DatePicker

The datepicker users jQuerie datepicker this will create a UI for an input. DatePicker will be used to enter in the purchase date and repair date. It is good to use the datepicker because it will make the user experience better, stop user error from happening because users can't enter dates in the wrong format.

Drap-and-Drop API

All elements in html are draggable you just have to enable them.The Drag and Drop feature will be used when selecting a courtesy phone. This will make the user experience more natrual and flowing as people prefer visual aids compared to just text.

Geolocation

Geolocation is an amazing tool it gets the users location using their ip address and mac address. It will be used to show the user where the phone fixing shop is relative to their position and how far away they are from the phone fixing shop.

Search Feature

Search feature is an amazing and needed tool it will search through an array or map or anything that is in a "list".Instead of the user scrolling through hundreds of FAQ's they can search for the one they are looking for and our app will display it so it will save our users time and increase users experience

Local Storage

We can use local storage which is built into all browsers to store username, other invoices and job Number. This will help store vital information like invoices and job number that we can auto increment.
